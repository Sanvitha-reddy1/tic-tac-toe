# Tic Tac Toe Game

A simple Tic Tac Toe game built using **HTML, CSS, and JavaScript**.
This project demonstrates basic DOM manipulation, event handling, and game logic using JavaScript.

## Features

* Two-player gameplay (Player X vs Player O)
* Interactive game board
* Automatic win detection
* Draw detection
* Reset game option

## Technologies Used

* **HTML5** – Structure of the game board
* **CSS3** – Styling and layout
* **JavaScript** – Game logic and interactivity

## How the Game Works

1. The game board consists of a 3×3 grid.
2. Players take turns clicking on empty cells.
3. The first player to align three symbols horizontally, vertically, or diagonally wins.
4. If all cells are filled and no player wins, the game ends in a draw.
5. The reset button allows players to start a new game.
